Demo stimuli

This app demonstrates typical stimulus/response experiment design.

The game runs completely in browser to preserve precise timing. Data is preloaded via js_vars and results are submitted via form.

Features:
  • Loading data from csv file
  • Pregenerating sequence of trials.
  • Waiting player to get ready.
  • Scheduling several display/input phases and pauses between trials.
  • Input by keys, screen touch or click.
  • Measuring reaction time.
  • Providing realtime feedback (wich is not secure).
  • Progress bar.
  • Custom data export.
